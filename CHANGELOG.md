# Changelog

All notable player-facing changes to Vivarium Libera are documented here.

This project is in early alpha; features and compatibility may change before the first stable release.

## Unreleased

### In development

- Multi-block alchemy table with a dedicated screen, renderer, recipes, and book integration.

## 0.1.0 — Alpha

### Added

- More than twenty medicinal, poisonous, decorative, wetland, and aquatic plants.
- Cuttable and regrowing plants plus the herbalist's knife.
- Animated mortar processing and the herbalist's reference book.
- Plum wood family, fruit, saplings, and compact tree generation.
- Placeable empty and filled water, oil, and wine jugs.
- Natural stream world-generation feature.
- English and Russian localization.
- GeckoLib animation support and optional Farmer's Delight compatibility.

### Compatibility

- Minecraft 1.21.1
- NeoForge 21.1.172 or newer 21.1.x
- GeckoLib 4.8.4 or newer
- Java 21
