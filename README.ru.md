# Vivarium Libera

**Vivarium Libera** — природно-травническое дополнение для Minecraft. Мод наполняет мир лекарственными растениями, болотной флорой, сливовыми деревьями, ручьями, ремесленными украшениями и первыми инструментами мастерской травника, сохраняя узнаваемый визуальный язык Minecraft.

> [!WARNING]
> Vivarium Libera находится в **ранней альфе**. Механики, рецепты, баланс и генерация мира могут меняться между сборками. Перед обновлением делайте резервную копию важных миров.

## Кратко о совместимости

| Компонент | Поддерживаемая версия |
| --- | --- |
| Minecraft | `1.21.1` |
| Загрузчик | NeoForge `21.1.172` или новее в ветке `21.1.x` |
| Java | `21` |
| GeckoLib | `4.8.4+` — обязателен |
| Farmer's Delight | Опциональная интеграция |
| Версия мода | `0.1.0` (альфа) |
| Mod ID | `vivariumlibera` |

Устанавливайте мод и GeckoLib и на клиент, и на сервер. Farmer's Delight необязателен: при его наличии Vivarium Libera добавляет совместимые рецепты нарезки и общие теги ингредиентов.

## Возможности

### Живой гербарий

- Более двадцати лекарственных, ядовитых, декоративных, болотных и водных растений.
- Отдельные правила размещения для обычной, влаголюбивой и водной флоры.
- Растения, которые можно срезать ножом травника и которые со временем отрастают.
- Русские и английские названия блоков, предметов, экранов и творческих вкладок.

### Ремесло травника

- Нож травника для аккуратной заготовки растительного сырья.
- Анимированная ступка с block entity для обработки травяных ингредиентов.
- Книга травника с внутриигровым экраном-справочником.
- Травяные порошки, яд и развивающийся набор мастерских рецептов.
- Многоблочный алхимический стол, который сейчас находится в активной разработке.

### Природа и декор

- Полный набор сливовой древесины: брёвна, обтёсанные брёвна, доски, ступеньки, плиты, заборы, калитки, кнопки, нажимные плиты, люки, листья и саженцы.
- Компактные сливовые деревья с несколькими стадиями листвы, спелыми и незрелыми плодами.
- Размещаемые глиняные кувшины для воды, масла и вина — пустые и наполненные.
- Природная генерация ручьёв, которая делает ландшафты менее статичными.

## Установка

1. Установите Minecraft `1.21.1` с NeoForge `21.1.172` или более новой версией из ветки `21.1.x`.
2. Установите GeckoLib `4.8.4` или новее для Minecraft `1.21.1`.
3. Скачайте Vivarium Libera на странице [GitHub Releases](https://github.com/iwosw/Vivarium-Libera/releases) или соберите мод из исходников.
4. Поместите `.jar`-файлы Vivarium Libera и GeckoLib в папку `mods` клиента и сервера.
5. При желании установите Farmer's Delight, чтобы включить рецепты совместимости.

## Сборка из исходников

Gradle Wrapper уже включён в проект — требуется только JDK 21.

```bash
# Windows
./gradlew.bat build

# Linux / macOS
./gradlew build
```

Готовый мод появится в `build/libs/`.

Полезные задачи для разработки:

```bash
./gradlew runClient
./gradlew runServer
./gradlew runData
```

## Структура проекта

| Путь | Назначение |
| --- | --- |
| `src/main/java/com/iwosw/vivariumlibera` | Игровая логика, реестры, клиентский код, интеграции и генераторы данных |
| `src/main/resources/assets/vivariumlibera` | Текстуры, модели, blockstates, анимации, геометрия и локализация |
| `src/main/resources/data` | Рецепты, теги, loot tables, data maps и данные генерации мира |
| `src/generated/resources` | Результаты генератора данных для сборок разработки |
| `docs` | Справочник травника, концепты и заметки по реализации |

## Поддержка и участие

Нашли баг или несовместимость? [Создайте issue](https://github.com/iwosw/Vivarium-Libera/issues) и приложите:

- точные версии Vivarium Libera, NeoForge, GeckoLib и Minecraft;

- последовательность действий для воспроизведения; и
- относящийся к проблеме фрагмент `latest.log` или crash report.

Исправления и точечные улучшения приветствуются. Старайтесь делать pull request небольшим, описывайте видимые игроку изменения и перед отправкой проверяйте проект командой `./gradlew build`.
