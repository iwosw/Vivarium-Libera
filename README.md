# Vivarium Libera

NeoForge-мод для Minecraft `1.21.1`, добавляющий декоративные растения, срезаемые травы и сливовую древесину.

## Что умеет мод сейчас

- Добавляет растения: полынь, крапиву, белену, зверобой, дурман, кипрей, цикорий, аир, рогоз, календулу, окопник, очанку, шалфей, вороний глаз, ликорис, кислицу двух цветов, тысячелистник четырёх цветов, колокольчики четырёх цветов и валериану трёх цветов.
- Растения имеют собственные модели, текстуры, blockstates, item-модели и loot tables.
- Полынь, крапива, белена, зверобой, дурман, кипрей и цикорий ставятся в один или два слоя через состояние `amount`.
- У stackable-растений есть направление `facing` и альтернативный визуальный вариант `alt`.
- Аир, календула, окопник, очанка, шалфей, вороний глаз и ликорис работают как простые декоративные bush-блоки.
- Кипрей, цикорий, рогоз, колокольчики и валериана используют двухблочные варианты.
- Кислица доступна в красном и жёлтом вариантах и ставится низким луговым слоем поверх блока.
- Тысячелистник доступен в белом, красном, розовом и жёлтом вариантах.
- Добавляет предмет `knife` с собственной текстурой и моделью; нож срезает цветущие растения, оставляя стебель до отрастания.
- Добавляет сливовую древесину: брёвна, обтёсанные брёвна, доски, ступеньки, плиты, кнопку, нажимную плиту, забор, калитку, люк, листья трёх стадий и саженец.
- Сливовый саженец вырастает в компактное дерево с коротким стволом и плотной кроной из первой стадии листвы без слив.
- Добавляет две creative-вкладки: растения Vivarium Libera и инструменты Vivarium Libera.
- Поддерживает русскую и английскую локализацию через `ru_ru.json` и `en_us.json`.

## Архитектура

- `src/main/java/com/iwosw/vivariumlibera/VivariumLibera.java` - точка входа мода.
- `src/main/java/com/iwosw/vivariumlibera/block` - переиспользуемое поведение блоков.
- `src/main/java/com/iwosw/vivariumlibera/registry` - регистрация блоков, предметов и creative-вкладок.
- `src/main/resources/assets/vivariumlibera` - модели, текстуры, blockstates и локализация.
- `src/main/resources/data/vivariumlibera` - игровые данные, включая loot tables.
- `src/main/resources/data/minecraft` - vanilla-теги для совместимости растений, листвы, брёвен и деревянных блоков.
- `src/main/resources/META-INF/neoforge.mods.toml` - metadata мода.

## Требования

- JDK `21`
- Gradle Wrapper: `gradlew` / `gradlew.bat`

## Команды

- `./gradlew build` - собрать мод.
- `./gradlew runClient` - запустить клиент для разработки.
- `./gradlew runServer` - запустить сервер для разработки.
